# Scroll Driven Hero Section Animation

## Live Demo
https://scroll-hero-animation-tbj5.vercel.app

## Project Overview
This project recreates a smooth and premium scroll-driven hero section inspired by modern web design trends. The animation is fully responsive and tied to scroll progress for a natural user experience.

## Tech Stack
- Next.js
- Tailwind CSS
- GSAP (ScrollTrigger)
- JavaScript
- HTML & CSS

## Features
- Smooth scroll-based animation
- Staggered headline reveal
- Sequential statistics animation
- Scroll-triggered image movement
- Responsive and performance-optimized design
- Premium UI and motion quality
